What's new in final version 1.6.8:

al ViruScan     
adalah salah satu aplikasi antivirus alternatif buatan anak bangsa indonesia yang ditujukan
untuk mendeteksi dan menghapus berbagai variant file virus,trojan,worm dan program malware lainnya.
Selain itu, aplikasi ini juga mempunyai built-in tools untuk membantu removal
virus/malware secara manual, saya mengharapkan agar aplikasi ini user-friendly
dan sifat-nya yang portable tanpa instalasi. Aplikasi ini dapat langsung dijalankan
langsung dari harddisk maupun dari flashdisk.
 
al ViruScan  
merupakan pengembangan dari Gee Are The Crying Machine yang dulu menggunakan
Cyclic Redundancy Check (CRC32) dalam mendeteksi file malware, 
sekarang menggunakan tehnik hashing satu arah Message-Digest Alghorithm 5 (MD5)
untuk mendeteksi file malware. 

al ViruScan v1.6.8(New)
dapat mencari dan menghapus semua file yang terinfeksi (virus/trojan) dengan cepat dan teliti.
dapat membunuh/menghapus semua jenis virus/trojan baik varian yang telah ada atau varian yang terbaru.
dapat memperbaiki kerusakan pada system yang biasa dilakukan oleh virus/trojan.
dapat dijalankan dari CD, Diskette, dan Flashdisk. (Tidak perlu diInstall)

	Features al VirusScan :
====================================================================	
- VirusScan Console
	- Semua konfigurasi al VirusScan terdapat pada VirusScan Console ini
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - VirusScan Console...  
- al VirusScan
	- Fasilitas ini user bisa menscan directori, drive, running process, autorun location,
	  removable disk atau komputer secara manual
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - al VirusScan... 
- VirusScan Registry Tweak v1.5
	- Fasilitas ini untuk mengembalikan beberapa setting windows yang telah
	  dirubah oleh virus, seperti menampilkan kembali Folder Options,
	  mengaktifkan Task Manager, Registry Editor dll.
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - VirusScan Console - Tools > VirusScan Registry Tweak...  
- VirusScan Process Manager v1.4
	- Fasilitas ini untuk melihat program-program yang sedang berjalan di windows,
	  bukan hanya melihat saja tetapi dapat pula men-suspend, resume dan menutup
	  (terminate) proses yang sedang berjalan serta dapat menscanning proses
	  yang sedang berjalan pada memory. dalam VirusScan Process Manager ini
	  juga terdapat fasilitas untuk melihat detail file dan melihat file dll yang di gunakan.
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - VirusScan Console - Tools > VirusScan Process Manager... 
- VirusScan Autorun Location 
	- Fasilitas ini digunakan untuk file-file pada lokasi autorun windows yang terdapat
	  di registry atau di startup directory.
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - VirusScan Console - Tools > VirusScan Autorun Location... 
- VirusScan Registry Editor
	- Fungsinya sama seperti Registry Editornya windows. kenapa ada 
	  VirusScan Registry Editor, karna dalam berbagai macam kondisi Registry
	  Editornya windows tidak dapat berjalan.	
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - VirusScan Console - Tools > VirusScan Registry Editor... 
- Default Scan 
	- Untuk melakukan pencarian pada drive atau direktory secara custom. 
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - al VirusScan...
- Scan Removable DIsk
	- Melakukan pencarian pada drive bertipe removable disk
	  contoh: Floppy Drive, Flash Disk dan lain-lain...
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - al VirusScan...
- Scan Computer
	- Pencarian file-file virus pada seluruh drive yang terdapat di komputer
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - al VirusScan...
- Scan Autorun locations
	- Mencari file-file pada lokasi autorun windows yang terdapat 
	  di registry maupun pada startup directory. Seperti 
	  'C:\windows\startup', RunOnce Shell dan lain-lain.
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - al VirusScan...
- Scan Running Process
	- Fasilitas ini untuk mencari file-file bervirus pada program yang sedang berjalan
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - al VirusScan...
- Set Attribute File Or Folder
	- Fasilitas ini untuk menampilkan file-file yang di disembunyikan (hide) oleh virus,
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - VirusScan Console - File > Set Attribute File Or Folder...
- Update Now
	- Dalam mengupdate virus, user dapat melakukan dengan dua cara
	  Yang pertama mendownload file update virus pada situs saya? ""
	  atau mengirimkan sample virus ke email saya.
	  sendiri Fasilitas ini masih dalam perbaikan... 
- Realtime Protection
	- Fasilitas ini untuk menjaga sistem komputer secara berkesinambungan.
	  user tidak perlu lagi menscan directory atau drive, karna al VirusScan sudah
	  bisa menscan secara otomatis.
- Auto Start On Startup
	- Secara default al VirusScan akan mensetting konfigurasi Auto Start On Startup...
	  menjadi enable
- Language English
	- Sudah diterapkan...
- Language Indonesia 
	- Fasilitas ini "Under Constraction"
- Virus Quarantine
	- Fasilitas ini untuk melihat virus apa saja yang berhasil ditangkap oleh al VirusScan,
	  user juga bisa mengembalikan virus ke asalnya... 
- Archive Zip 
	- Fasilitas ini "Under Constraction"
- Archive Rar 
	- Fasilitas ini "Under Constraction"
- Running On Safe Mode
	- Fasiltas ini al VirusScan akan tetap bisa berjalan pada sistem Safe Mode 
- On Access Registry
	- Semua konfigurasi al VirusScan disimpan dalam registry
- Send The Example Of Virus
	- User bisa menggunakan fungsi ini untuk mengirimkan sample virus yang
	  belum dikenali ke email saya
- Scan With Virus Sample (New) v1.4
	- Gunakan menu al VirusScan Scan With Virus Sample untuk menghapus virus / trojan yang 
	  menginfeksi Komputer pada menu ini Anda akan diminta memilih satu file yang 
	  diyakini sebagai virus pada sistem komputer.
	  Saat pemilihan file Anda harus yakin bahwa file yang dipilih merupakan virus 
	  / Trojan.
	  user dapat menggunakan fasilitas ini dengan cara klik icon al VirusScan
	  pada sytem tray - al VirusScan Scan With Sample...
- Fix Registry (New)
	- Gunakan menu ini untuk memperbaiki sistem windows yang telah dirubah oleh virus,
	  fix registry telah bisa memperbaiki beberapa setting windows yang telah dirubah 
	  oleh virus :
	  	4k51k4
	  	Amburadul.Hokage Killer
	  	Flu_Ikan
		etc...
